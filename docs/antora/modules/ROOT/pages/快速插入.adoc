= 快速插入大量数据

向 MySQL 数据库中，快速插入大量数据。

== 合并语句

参考 https://dev.mysql.com/doc/refman/5.7/en/insert-optimization.html

合并语句以减少应用和数据库之间的传输时间。

.常规语句
[source]
----
INSERT INTO `table1` (`field1`, `field2`) VALUES ("data1", "data2");
INSERT INTO `table1` (`field1`, `field2`) VALUES ("data1", "data2");
INSERT INTO `table1` (`field1`, `field2`) VALUES ("data1", "data2");
INSERT INTO `table1` (`field1`, `field2`) VALUES ("data1", "data2");
INSERT INTO `table1` (`field1`, `field2`) VALUES ("data1", "data2");
----

.合并后语句
[source]
----
INSERT INTO `table1` (`field1`, `field2`) VALUES ("data1", "data2"),
                                                 ("data1", "data2"),
                                                 ("data1", "data2"),
                                                 ("data1", "data2"),
                                                 ("data1", "data2");
----
